Text to speech is a program using gTTS library to convert text into speech.

This program is capable of converting the text written by the user into speech.
The user first needs to run the program and then the program will be waiting for the
user input in the file "fl.txt"(Attached),once the user gives input into the fl.txt and saves the file
the program will automatically read out the text.

The program wont go into loop of reading out the text again and again but will wait for the 
user to write something new in order to read out the text.
